## ✨ Features

### 🏢 Community & Tribes (New!)
- **Create Your Own Community**: Launch a specialized running tribe for your city, night runs, or trail adventures.
- **Real-Time Chat**: Stay connected with members through persistent community chat rooms.
- **Discover Tribes**: Browse through active groups and join ones that match your pace and personality.

### 🏆 Marathon Management
- **Host Events**: Create custom marathons with prize pools, locations, and descriptions.
- **Participant Tracking**: Manage sign-ups and declare winners with ease.
- **Relive the Momentum**: Upload and browse through post-race galleries with high-octane photography.

### 📊 Performance & Social
- **Global Leaderboard**: See where you rank among the world's most consistent athletes.
- **Strava Sync**: Connect your elite fitness accounts to automatically pull your running data.
- **Modern Dashboard**: A sleek, mobile-responsive profile to track your hosted events and victories.

## 🛠️ Tech Stack

- **Frontend**: [React 19](https://react.dev/), [Vite 6](https://vitejs.dev/), [Tailwind CSS 4](https://tailwindcss.com/)
- **Backend**: [Express](https://expressjs.com/), [tsx](https://github.com/privatenumber/tsx)
- **Database**: [SQLite](https://www.sqlite.org/) via `better-sqlite3`
- **Animations**: [Motion](https://motion.dev/) (Framer Motion)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (LTS recommended)
- [npm](https://www.npmjs.com/)

### Installation

1. **Clone the repository:**
   ```bash
   git clone <your-repo-url>
   cd runhost
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Configure Environment:**
   Create a `.env` file (copy from `.env.example`) and add your keys:
   ```env
   JWT_SECRET=your_secret_key
   STRAVA_CLIENT_ID=your_id
   STRAVA_CLIENT_SECRET=your_secret
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```
   The app will be available at `http://localhost:3000`.
